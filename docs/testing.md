# 测试与验收

## 测试分层

- 单元测试：真太阳时计算、子时索引、地理解析、歧义处理。
- 集成测试：从输入参数到 `iztro` 排盘结果的全链路。
- 样例回归：固定出生样例，校验关键字段不漂移。
- CLI 验收：命令行文本输出、JSON 输出、歧义报错。

## 本地执行

```bash
npm test
npm run test:cli
npm run chart -- --date 1990-05-15 --time 14:30 --gender male --place 北京市东城区
npm run chart -- --date 1988-02-09 --time 23:40 --gender female --place 杭州市余杭区 --json
```

## 验收清单

- [x] 能按区县级地点解析经纬度。
- [x] 地点歧义时会提示候选，不默默猜测。
- [x] 真太阳时同时考虑经度修正与时间方程。
- [x] 早子时 / 晚子时正确区分为 `timeIndex=0/12`。
- [x] 能输出文本报告与 JSON 结构化结果。
- [x] OpenClaw skill 调用入口保持可用。
- [x] 本地测试可完整跑通。

## 当前最小地理数据覆盖

- 覆盖全国 34 个省级行政区的代表性区县。
- 额外补充北京、上海、重庆、苏州吴江、杭州余杭、深圳宝安等高频出生地。
- 当前仍非全国全量区县库；后续可按同结构扩充到完整民政部/统计局行政区数据。
