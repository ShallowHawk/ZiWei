# 紫微斗数理论速查

本文件用于给上层模型做报告时提供一个简明、可扩展的理论框架。

## 1. 使用原则

- 星曜看人格底色
- 宫位看人生领域
- 四化看事件触发
- 三方四正看整体结构
- 真太阳时优先于未经修正的钟表时间

## 2. 门派整合建议

### 三合派
重点看：
- 命、财、官、迁四宫
- 主星组合
- 格局感与整体人生走向

适合用于：
- 做整体基调判断
- 描述人生画面感
- 解释事业、财运、社会位置

### 四化派
重点看：
- 化禄
- 化权
- 化科
- 化忌

适合用于：
- 解释事情为何触发
- 分析矛盾点、执念点、风险点
- 为流年/大限扩展做基础

### 中州派
重点看：
- 星系互涉
- 宫位结构
- 借宫、转宫、整体联动

适合用于：
- 修正单宫单星武断解读
- 做结构级解释
- 平衡三合派与四化派

## 3. 报告生成建议

推荐输出顺序：

1. 先给基本盘信息
2. 再给命宫和三方四正
3. 再给财帛、官禄、夫妻等重点宫位
4. 再说明潜在课题与建议
5. 若用户确认，再继续做未来运势

## 4. 敏感议题表达原则

- 不使用恐吓式语言
- 不对疾病、寿命做武断断言
- 婚姻、财富、健康结论只做趋势提醒
- 始终给出可执行建议

## 5. 工程上的边界

当前仓库先解决：
- 输入标准化
- 真太阳时修正
- `iztro` 排盘
- 宫位与星曜结构化输出

后续再追加：
